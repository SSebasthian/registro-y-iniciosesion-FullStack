<h1>🚀 Proyecto Fullstack (Angular + Spring Boot)</h1>

<p>
Aplicación web ecommerce con autenticación, gestión de usuarios, roles y permisos
</p>



<h2>📋 Descripción</h2>
<p>
Aplicación web ecommerce desarrollada con <b>Angular 19</b> en el frontend, 
<b>Spring Boot</b> en el backend y <b>MySQL</b> en base de datos.  
<br>
Permite el registro, inicio de sesión y la gestión de usuarios, roles y permisos.
</p>

<h2>📥 Instalación</h2>

<h3>1. Clonar repositorio</h3>

<pre>
git clone https://github.com/SSebasthian/registro-y-iniciosesion-FullStack.git
</pre>
<i>💡 Puedes cambiar el nombre de las carpetas si lo deseas.</i>
<br><br>

<h3>2. Base de Datos (MySQL)</h3>

<ol>
  <li>Abrir MySQL Workbench</li>
  <li>Crear base de datos:</li>
  <li><pre>CREATE DATABASE nombre_bd;</pre></li>
</ol>
<br>

<h3>3. Backend (Spring Boot)</h3>
<ol>
  <li>Entrar al backend<pre>cd backend</pre></li>
  <li>Instalar dependencias<pre>mvn clean install</pre></li>
  <li>Configurar proyecto<pre>Cambiar nombres si es necesario (pom.xml, package, clase principal)</pre></li>
  <li>Abrir en IntelliJ IDEA</li>
  <li>Configurar base de datos<pre>src/main/resources/application.properties</pre></li>
  <li>Ejecutar<pre>Ejecutar la clase principal del proyecto (Application.java)</pre></li>
</ol>
<br>

<h3>4. Frontend (Angular)</h3>
<ol>
  <li>Entrar al frontend<pre>cd frontend</pre></li>
  <li>Instalar dependencias<pre>npm install</pre></li>
  <li>Abrir Proyecto con Visual Studio Code<pre>code .</pre></li>
  <li>Ejecutar<pre>ng serve</pre></li>
</ol>
<br>


<h2>▶️ Ejecución</h2>

<ul>
  <li>🔗 Backend: http://localhost:8080</li>
  <li>🌐 Frontend: http://localhost:4200</li>
</ul>
